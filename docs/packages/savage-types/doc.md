# Savage-types

> A library for determining the type of JavaScript variables. It is simple, convenient, and can be used out of the box. It provides many methods for use.

## Installtion

::: code-group

```[npm]
npm i savage-types
```

```[pnpm]
pnpm add savage-types
```

```[yarn]
yarn add savage-types
```

:::
