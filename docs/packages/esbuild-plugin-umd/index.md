# Esbuild-plugin-umd

> A esbuild plugin for building UMD module.

## Installtion

::: code-group

```[npm]
npm i esbuild-plugin-umd
```

```[pnpm]
pnpm add esbuild-plugin-umd
```

```[yarn]
yarn add esbuild-plugin-umd
```

:::
