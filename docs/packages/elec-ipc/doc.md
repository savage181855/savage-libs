# Elec-ipc

> A library that makes communication between different processes more convenient for Electron applications.

## Installtion

::: code-group

```[npm]
npm i elec-ipc
```

```[pnpm]
pnpm add elec-ipc
```

```[yarn]
yarn add elec-ipc
```

:::
