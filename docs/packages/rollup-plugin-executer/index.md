# Rollup-plugin-executer

> A rollup plugin can execute commands or functions on a specified hook.

## Installtion

::: code-group

```[npm]
npm i rollup-plugin-executer
```

```[pnpm]
pnpm add rollup-plugin-executer
```

```[yarn]
yarn add rollup-plugin-executer
```

:::
